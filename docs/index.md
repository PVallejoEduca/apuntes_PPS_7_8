# 📱 Desarrollo Android Profesional (FP 2026)

¡Hola! Soy Mikel Jorge, tu profesor del ciclo de Desarrollo de Aplicaciones Multiplataforma (DAM) en Navarra. Me apasiona la programación móvil y mi objetivo es explicarte la tecnología de la forma más práctica, precisa y directa posible. 

He creado este portal para que sea tu manual de supervivencia, tu principal herramienta de consulta y tu mapa de ruta durante todo el curso.

!!! success "¿Qué pretendemos con esta web?"
    El objetivo es transformarte en un desarrollador Android moderno y preparado para el mercado laboral actual. Aquí no vamos a perder el tiempo con tecnologías obsoletas. 
    
    Vamos a aprender a construir aplicaciones nativas, rápidas y escalables utilizando el estándar absoluto de la industria: **Kotlin** y **Jetpack Compose**. Cambiaremos tu forma de pensar (del paradigma imperativo al funcional), dominaremos la arquitectura profesional y terminaremos subiendo tu trabajo a la Play Store.



---

## 🛠️ Nuestra Metodología (Cómo vamos a trabajar)

Este no es un curso de leer y memorizar; es un curso de picar código. Para asegurar que asimilas cada concepto, la web está estructurada bajo estas reglas:

* **💻 Código 100% Práctico:** Todas las lecciones teóricas están acompañadas de ejemplos de código reales, comparativas visuales y animaciones para que veas el resultado exacto en el dispositivo.
* **🧩 22 Micro-prácticas:** El curso consta de 22 módulos. Al final de cada uno de ellos, te enfrentarás a un pequeño reto guiado (formato *Codelab*) para asentar exclusivamente el concepto que acabamos de aprender.
* **🏗️ 6 Macro-proyectos de Bloque:** Al finalizar cada uno de los 6 grandes bloques del curso, construiremos un proyecto real y completo desde cero que integrará todo lo aprendido en esas semanas (por ejemplo, una app del tiempo conectada a internet o un gestor con base de datos).
* **🐙 Todo en GitHub:** Cero archivos `.zip` sueltos por el aula. Trabajaremos con repositorios profesionales. Te proporcionaré código base (`starter-code`) para ahorrar tiempo de configuración y código resuelto (`solution-code`) para que puedas autocorregirte.

  </a>

  <a href="bloque_6/b6-inicio" class="course-card">

    <p><strong>Módulos 21 y 22.</strong> El último paso antes de la gloria. Aprenderemos a testear nuestro código para evitar "crasheos" en producción, ofuscaremos el código por seguridad y generaremos la versión final para publicarla en la Play Store.</p>
  </a>
